# JARVIS Audio Placeholders

These WAV files are silent placeholders so the Phase 1 audio pipeline can be tested before final sound design.

- `boot_intro.wav`: short cinematic boot cue.
- `listening.wav`: quick attentive chime after wake word detection.
- `working.wav`: subtle processing tick or low pulse.
- `done.wav`: short completion chime after speech.
- `error.wav`: restrained failure tone.
